# Changelog

All notable changes to Misi Screener will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-24

### Added
- Autonomous trading engine with HedgeFundMasterAgent
- YAML-based strategy definition system
- "No Gimmick" backtesting engine
- 9 specialized trading agents (Signal, Portfolio, Risk, Technical, Fundamental, Sentiment, Trader, Orchestrator, Master)
- 12 analytical component engines (Quant Scoring, Market Structure, Liquidity, DEX Intelligence, etc.)
- Real-time data integration (Yahoo Finance, Alpha Vantage, NewsAPI)
- Interactive web dashboard with FastAPI backend
- AI Query Orchestrator for natural language analysis
- Quant Scoring Engine for unified quantitative assessment
- Paper trading broker for simulated execution
- Comprehensive test suite

### Changed
- Migrated license from Unlicense to MIT

### Security
- Added CONTRIBUTING.md, CODE_OF_CONDUCT.md, SECURITY.md
- Added GitHub issue templates and PR template
